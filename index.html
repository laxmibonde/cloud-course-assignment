// Cloud Evolution Timeline

Mainframe Era

1960s – 1970s

Centralised computing where users shared mainframe resources via dumb terminals. The earliest form of shared computing infrastructure.

Client-Server Model

1980s – 1990s

PCs and local networks shifted computing to distributed systems. Servers hosted applications and databases while clients ran the UI.

Virtualisation Revolution

Early 2000s

VMware introduced enterprise virtualisation, allowing multiple operating systems on one physical machine — laying the groundwork for cloud.

Birth of Public Cloud

2006 — AWS Launches EC2 & S3

Amazon Web Services launched EC2 and S3, allowing anyone to rent compute and storage on demand. The modern cloud era begins.

Cloud Goes Mainstream

2010s — Azure, GCP, Docker

Microsoft Azure and Google Cloud Platform expand the market. Docker (2013) revolutionises containerisation. Serverless computing emerges.

Cloud-Native & Edge

2020s — Present

Kubernetes, service meshes, and edge computing push workloads closer to users. AI/ML workloads become a primary driver of cloud adoption.
